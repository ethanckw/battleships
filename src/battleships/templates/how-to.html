How To

A bot is a file that is able to play a game of Battleships.

Each time it's the bot's turn to make a move the bot file will be executed and passed the current board state as standard input (stdin).

A board state is a 10-by-10 grid:

   {% for l in ["A", "B", "C", "D", "E", "F", "G", "H", "I", "J"] %} {{ l }} {% end %}
{% for y in range(10) %}{% set i = y+1 %}{% raw (" %s" if i < 10 else "%s") % i %} {% for x in range(10) %}{% set i = ((y * 10) + x) %}{% raw (" %s" if i < 10 else "%s") % i %} {% end %}
{% end %}

It's passed to the bot file as a single comma-separated string.

The numbers in the grid show the index positions of the squares in this comma-separated list. Each square has one of the following values: 

1 - Hit
0 - Unknown
-1 - Miss

The string received by the bot might look like: 

-1,0,0,1,0,0...

The bot can look at the board state and must return the index position of the square it would like to try next.

The bot might decide it wants to try square C5 next so it will return 42 (see grid above).

To return the value the bot file must write it to standard out.

That's it! The bot will be repeatedly executed like this until all battleships have been sunk. The bot's score will be based on the number of turns it took.

Samples

The following bots are written in various languages but all implement the same simple strategy. First they try square A1, then B1, then C1, scanning the board until eventually all battleships are sunk.

Python

#!/usr/bin/python
import sys

def main():
shot_grid = sys.argv[1].split(',')
for i, state in enumerate(shot_grid):
    if state == "0": # unknown
        sys.stdout.write(str(i))
        return

if __name__ == "__main__":
main()

Bash

#!/bin/bash

shot_grid=(${1//,/ })
for i in "${!shot_grid[@]}"; do
state="${shot_grid[$i]}"
if [ "$state" == "0" ]; then # unknown
    echo "$i"
    exit
fi
done

PHP

#!/usr/bin/php

<?php
$shot_grid = $argv[1];
$shot_grid = explode(",", $shot_grid);
foreach ($shot_grid as $i => $state) {
    if ($state == "0") { // unknown
        echo $i;
        exit;
    }
}
?>
{% end %}
